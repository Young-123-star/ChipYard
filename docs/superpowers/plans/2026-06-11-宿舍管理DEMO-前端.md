# 宿舍管理系统 DEMO — 前端实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 用 Vue3 + Vite + Element Plus 实现宿舍资源管理 DEMO 的 Web 管理后台：登录 + 后台外壳（侧边栏/顶栏）+ 楼栋/楼层/房间/床位 CRUD + 房间状态看板，对接已完成的 Spring Boot 后端（:8080）。

**Architecture:** Vite 单页应用。`utils/request.ts` 封装 axios（统一注入 JWT、解析 `R{code,message,data}` 信封、统一错误提示、401 跳登录）。Pinia 存 token/用户信息，Vue Router 做登录守卫。`api/` 按模块封装请求，`views/` 按模块组织页面，`layout/` 提供后台外壳。前端不做单元测试，按 DEMO 约定以"启动 + 手动点测"验收（每个任务给出明确的手动验证步骤）。

**Tech Stack:** Node 24 / npm 11、Vue 3.4、Vite 5、Element Plus 2.x、Pinia 2、Vue Router 4、Axios 1.x、TypeScript 5。

**后端契约（已定型，见 dms-backend）：**
- 统一返回 `{ code, message, data }`，`code===0` 成功；分页 `data = { records, total, page, size }`。
- 登录 `POST /api/auth/login {username,password}` → `data {token,userId,username,realName}`；`GET /api/auth/me`。
- 楼栋 `GET /api/buildings?buildingName=&status=&page=&size=`（分页）、`GET/POST/PUT/DELETE /api/buildings(/{id})`。
- 楼层 `GET /api/floors?buildingId=`（列表）、`POST/PUT/DELETE /api/floors(/{id})`。
- 房间 `GET /api/rooms?buildingId=&floorId=&roomType=&status=&page=&size=`（分页）、`GET/POST/PUT/DELETE /api/rooms(/{id})`、`GET /api/rooms/board?buildingId=&floorId=`。
- 床位 `GET /api/beds?roomId=`（列表）、`POST/PUT/DELETE /api/beds(/{id})`。
- 非登录接口需请求头 `Authorization: Bearer <token>`；401 时后端返回 HTTP 401 + JSON。
- 枚举：房型 1单/2双/3四/4六/5八/6其他；房间状态 0停用/1空闲/2已满/3维修中/4预留；床位状态 0停用/1空闲/2已入住/3维修中/4预留；性别限制 0不限/1男/2女；床位类型 1上床/2下床/3单床。
- 种子账号 admin/admin123。

**约定：** 前端代码在 `dms-frontend/`，分支沿用 `feat/demo-backend`。每个任务结束 commit（提交信息前缀 `feat(fe):` 等）。Windows 环境，Node/npm 已装。

- [ ] **Step 2: 创建 `src/utils/request.ts`（拦截器：注入 token、解析信封、错误提示、401 跳登录）**

```ts
import axios from 'axios'
import { ElMessage } from 'element-plus'
import { useUserStore } from '@/stores/user'
import router from '@/router'

const service = axios.create({
  baseURL: '/api',
  timeout: 10000
})

service.interceptors.request.use((config) => {
  const userStore = useUserStore()
  if (userStore.token) {
    config.headers.Authorization = `Bearer ${userStore.token}`
  }
  return config
})

service.interceptors.response.use(
  (response) => {
    const res = response.data
    if (res.code === 0) {
      return res.data
    }
    ElMessage.error(res.message || '请求失败')
    return Promise.reject(new Error(res.message || 'Error'))
  },
  (error) => {
    if (error.response && error.response.status === 401) {
      const userStore = useUserStore()
      userStore.logout()
      router.push('/login')
      ElMessage.error('登录已过期，请重新登录')
    } else {
      ElMessage.error(error.message || '网络错误')
    }
    return Promise.reject(error)
  }
)

export default service
```

> 说明：响应拦截器在 `code===0` 时直接返回 `data`，因此 api 函数的返回类型就是 `data` 的类型。注意 `request.ts` 依赖 `@/stores/user` 和 `@/router`（Task 3 创建）——Task 3 完成后整体可编译；本任务先把文件建好，编译验证放在 Task 3 末尾。

- [ ] **Step 1: 创建 `src/stores/user.ts`**

```ts
import { defineStore } from 'pinia'
import { ref } from 'vue'
import type { CurrentUser } from '@/api/auth'

export const useUserStore = defineStore('user', () => {
  const token = ref<string>(localStorage.getItem('dms_token') || '')
  const userInfo = ref<CurrentUser | null>(null)

  function setToken(t: string) {
    token.value = t
    localStorage.setItem('dms_token', t)
  }

  function setUserInfo(u: CurrentUser) {
    userInfo.value = u
  }

  function logout() {
    token.value = ''
    userInfo.value = null
    localStorage.removeItem('dms_token')
  }

  return { token, userInfo, setToken, setUserInfo, logout }
})
```

- [ ] **Step 2: 创建 `src/router/index.ts`（路由表 + 登录守卫）**

```ts
import { createRouter, createWebHistory, type RouteRecordRaw } from 'vue-router'
import { useUserStore } from '@/stores/user'

const routes: RouteRecordRaw[] = [
  { path: '/login', name: 'login', component: () => import('@/views/login/index.vue') },
  {
    path: '/',
    component: () => import('@/layout/index.vue'),
    redirect: '/buildings',
    children: [
      { path: 'buildings', name: 'buildings', meta: { title: '楼栋管理' }, component: () => import('@/views/resource/building.vue') },
      { path: 'floors', name: 'floors', meta: { title: '楼层管理' }, component: () => import('@/views/resource/floor.vue') },
      { path: 'rooms', name: 'rooms', meta: { title: '房间管理' }, component: () => import('@/views/resource/room.vue') },
      { path: 'beds', name: 'beds', meta: { title: '床位管理' }, component: () => import('@/views/resource/bed.vue') },
      { path: 'board', name: 'board', meta: { title: '房间状态看板' }, component: () => import('@/views/resource/board.vue') }
    ]
  }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

router.beforeEach((to, _from, next) => {
  const userStore = useUserStore()
  if (to.path === '/login') {
    next()
  } else if (!userStore.token) {
    next('/login')
  } else {
    next()
  }
})

export default router
```

  formFloors.value = []

}

  formFloors.value = await listFloors(row.buildingId)
  dialogVisible.value = true
}

  saving.value = true
  try {
    if (form.id) {
      await updateRoom(form.id, form)
    } else {

    }
    ElMessage.success('保存成功')
    dialogVisible.value = false
    reload()
  } finally {
    saving.value = false
  }
}
