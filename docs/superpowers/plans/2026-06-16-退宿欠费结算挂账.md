# 退宿欠费结算挂账 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 办理退宿时在同一事务内把该在住档案的未缴账单结算挂账——账单置「挂账」、退宿单回填「离场欠费金额」，退宿不阻断；挂账账单后续可缴清。

**Architecture:** checkout → fee 单向依赖。`FeeBillService` 新增 `listUnpaidByRecord` / `settleArrearsForRecord`，并放宽 `pay` 守卫（未缴或挂账可缴）。`CheckoutServiceImpl.confirm` 末尾调 `settleArrearsForRecord` 并把欠费总额写入退宿单（同一 `@Transactional`）。新增只读 arrears 预览接口供前端弹窗。

**Tech Stack:** Spring Boot 3.2 / JDK 17 / MyBatis-Plus / Spring Security+JWT / H2；前端 Vue3 + Vite + Element Plus + Pinia + Axios。

设计依据：`docs/superpowers/specs/2026-06-16-退宿欠费结算挂账-design.md`。

追加两方法：
```java
    @Override
    public java.util.List<FeeBill> listUnpaidByRecord(Long checkinRecordId) {
        return billMapper.selectList(Wrappers.<FeeBill>lambdaQuery()
                .eq(FeeBill::getCheckinRecordId, checkinRecordId)
                .eq(FeeBill::getStatus, 1));
    }

    @Override
    public java.math.BigDecimal settleArrearsForRecord(Long checkinRecordId) {
        java.util.List<FeeBill> unpaid = listUnpaidByRecord(checkinRecordId);
        java.math.BigDecimal total = java.math.BigDecimal.ZERO;
        for (FeeBill b : unpaid) {
            total = total.add(b.getAmount());
            b.setStatus(4);
            billMapper.updateById(b);
        }
        return total;
    }
```

放宽 `pay` 的状态守卫，把：

（`voidBill` 的 `if (b.getStatus() != 1)` 保持不变。）

`confirm` 方法体改为（在档案归档之后、退宿单完成时把欠费总额写入）：
```java
    @Override
    @Transactional
    public void confirm(Long orderId, LocalDate checkoutDate) {
        CheckoutOrder order = getOrder(orderId);
        if (order.getStatus() != 1) throw new BizException("仅待退宿的单可办理退宿");
        if (order.getCheckinRecordId() == null) throw new BizException("该退宿单无关联在住档案，无法办理");

        CheckinRecord record = checkinService.getRecord(order.getCheckinRecordId());
        if (record.getStatus() != 1) throw new BizException("关联入住档案非在住状态");

        // 释放床位 + 刷新房间统计
        bedService.release(record.getBedId());
        roomService.refreshOccupancy(record.getRoomId());

        // 档案归档
        checkinService.markCheckedOut(record.getId(), checkoutDate != null ? checkoutDate : LocalDate.now());

        // 欠费结算挂账：未缴账单 → 挂账，回填离场欠费金额
        order.setArrearsAmount(feeBillService.settleArrearsForRecord(record.getId()));

        // 退宿单完成
        order.setStatus(2);
        orderMapper.updateById(order);
    }
```
