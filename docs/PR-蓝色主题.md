# PR 描述 — 蓝色企业主题 + 深海军蓝侧栏

> 用于 `feat/ui-blue-theme` → `main` 的合并请求。

## 标题

```
feat: 全局蓝色企业主题 + 深海军蓝实色侧栏（ChipMore Dorm）
```

## 概述

将管理后台的整体视觉从「Apple 毛玻璃浅色风」调整为「蓝色企业主题 + 深海军蓝实色侧栏」，参考 P2P 采购系统的后台风格。**仅改动全局样式令牌与外壳布局**，所有业务页面（楼栋/楼层/房间/床位/看板）通过共享的 `--dms-*` CSS 变量自动换肤，零逐页改动。登录页保持原 Apple 风。

## 主要内容

### 1. 全局主题令牌（`src/styles/theme.css`）
- 主题色改蓝：`--dms-accent: #1f6feb`，同步覆盖 Element Plus `--el-color-primary` 系列
- 内容区改冷调浅灰底、毛玻璃卡 → 干净白卡（去 `backdrop-filter`，改细边框 + 柔和投影）
- 新增深蓝侧栏色板 `--dms-nav-*`：top `#0c2f63` / bottom `#0a2750` / 选中 `#1f6feb`
- 表格表头底色、行悬浮色、状态色微调以协调蓝主题
- **保留所有 `--dms-*` 变量名**，业务页面无需改动即自动适配

### 2. 后台外壳（`src/layout/index.vue`）
- 侧栏：深海军蓝实色渐变 + 白色文字，品牌处加圆角 logo 块「C」
- 菜单按 `el-menu-item-group` 分组（资源管理 / 视图），层次更清晰
- 选中项亮蓝填充高亮块 + 柔光；顶栏改纯白

### 3. 登录页
- 维持原 Apple 风（字标渐变 + 芯片电路 SVG 背景 + 毛玻璃登录卡）
- 登录按钮随主题色自然变蓝

### 4. 文档 / 留档
- P2P 系统风格参考图 10 张入库 `docs/style-previews/P2PStyle*.png`
- 静态示例 `blue-theme-preview.html` + 真机验证截图 `blue-applied-{buildings,board}.jpeg`

## 验证

真机启动前后端实测：登录 → 楼栋（卡片）/ 楼层（表格）→ 房间状态看板，全部正常渲染，浏览器控制台零报错。

## 不在本次范围

业务功能无改动（纯视觉）；二期业务模块、MySQL/Docker 部署等仍未涉及。
